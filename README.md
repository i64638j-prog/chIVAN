# Weather Diary - Дневник погоды

## Автор
**Иван Петров** (Ivan Petrov)

## Описание программы
Weather Diary - это графическое приложение на Python для ведения дневника наблюдений за погодой. Программа позволяет:

- Добавлять записи о погоде с указанием даты, температуры, описания и наличия осадков
- Просматривать все записи в удобной табличной форме
- Фильтровать записи по дате и температуре
- Сохранять данные в JSON-файл и загружать из него
- Проверять корректность вводимых данных

## Требования
- Python 3.6 или выше
- Стандартные библиотеки Python (tkinter, json, datetime) - не требуют дополнительной установки

## Как запустить приложение

1. Скачайте файл `weather_diary.py`
2. Откройте терминал/командную строку в папке с файлом
3. Выполните команду:
   ```bash
   python weather_diary.py
